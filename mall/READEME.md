### 安装更新wepy命令行工具
```
npm install wepy-cli -g
```
### 生成项目
```
wepy init standard mall
```
### 安装依赖
```
cd myproject
npm install
```
### 开发实时编译
```
wepy build --watch
```
### 开发者工具导入项目
使用微信开发者工具新建项目，本地开发选择项目根目录，会自动导入项目配置。
